# 🩺 CureConnect – AI Powered Smart Healthcare Platform

CureConnect is an innovative healthcare mobile application designed to improve medical accessibility, emergency response, and preventive healthcare using AI-driven technologies.
The platform connects patients, hospitals, donors, and emergency services into one intelligent ecosystem.

---

## 🚀 Features

*  **AI-Based Disease Prediction**
  Predict possible diseases based on symptoms using intelligent AI models.

*  **Smart Hospital Recommendation System**
  Find the most suitable nearby hospitals based on requirements and availability.

*  **Medical Petition Application**
  Users can submit petitions for medical support and urgent healthcare assistance.

*  **Emergency Services Integration**
  Quick emergency access and response support for critical situations.

*  **Blood Donation & Donor Discovery**
  Connect blood donors with patients in real-time during emergencies.

*  **HealthScore for Preventive Care**
  Personalized health scoring system to encourage preventive healthcare habits.
